# Screenshots

Automatisch erzeugte Full-Page-Screenshots aller Live-Seiten der MH3-Webseite. Pro Seite zwei Dateien: `<seite>-desktop.png` (1440 px Breite) und `<seite>-mobile.png` (375 px Breite).

Regenerierung über `pnpm screenshots`. Skript: `scripts/screenshots.mjs`. Konfiguration und Lauflogik sind dort dokumentiert.
