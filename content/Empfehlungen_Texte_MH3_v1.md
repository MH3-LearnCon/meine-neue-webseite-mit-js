## Hinweise für Schritt F (Cursor)

**Komponenten-Status:**
- Keine ContactSection auf dieser Seite.
- Buch-Karten-Pattern ist NEU im Design-System — Komponente `BookRecommendationCard.tsx` erstellen. Vorlage: einzeilige breite Karte mit Border und shadow-sm, eine pro Zeile, volle Containerbreite. Autor klein oben, Titel als visuell starkes Element, Platzhalter-Satz dezent darunter, Sprach-Buttons rechts oder unten.
- Sprach-Buttons als sekundäre Buttons (Denver-Blau-Border oder ähnlich, Hover-Konvention: `hover:bg-denver-blue hover:text-white hover:border-denver-blue`).
- Affiliate-Hinweis „(bezahlter Link)" entweder als kleiner Suffix neben dem Button oder als Fußnoten-Indikator — Cursor entscheidet pragmatisch im Bau.

**Hintergrund-Alternierung:**
- Erste Section (H1 + Lead) auf `bg-[#F5F5F5]` (analog `/kein-coaching`).
- Folgende Sections im Wechsel — Cursor übernimmt das Pattern wie auf den Bereichsseiten.

**Schema.org:**
- ItemList am Seiten-Anfang im JSON-LD-Block.
- Pro Buch ein `Book`-Eintrag mit `name`, `author`, `inLanguage`.
- Affiliate-Hinweise im strukturierten Markup nicht nötig (das ist Pflicht-Text auf der Seite, kein Schema-Feld).

**Bewusst NICHT auf der Seite:**
- Keine Buch-Cover-Bilder (DSGVO + Amazon-Lizenz + Urheberrecht).
- Keine ContactSection.
- Keine eigene Karriere-Kategorie (mangels Buch).
- Kein externer Link auf das Kinderhospiz.
- Keine englische Version der Einleitung (nur Link-Buttons in beiden Sprachen).

**Tote Querlinks geschlossen:**
